(Assignment) Write a Shell

Submit your code as a single tar.gz file called MIS.tar.gz.

Assignment: Write a Shell

Write a shell. The shell should have minimum following features:

  • Read commands in a loop,
  • show prompt,
  • run commands,
  • run commands in background,
  •  > (output redirection),
  • < (input redirection) and
  • | (pipe)
  • Handle signals Ctrl-C and Ctrl-z
  • fg , bg

Notes:

You will have to learn following system calls, to a minimum, to write your shell.

fork

exec with all variants like excel, execvp, execlp, etc.

open

write

read

close

pipe

dup

signal

kill

sigaction
